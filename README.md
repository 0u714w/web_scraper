# Web Scraper

Program to scrape links, email addresses, phone numbers, and images from a website

## How to do it

Clone this repository
```
git clone https://github.com/dougenas/web_scraper.git
```

## Run the program
```
python scraper.py <url>
```

Happy hacking!
